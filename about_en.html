About and Legal Notice

This website follows a digital sobriety approach. It is coded only in HTML and CSS and uses very few images in order to reduce resource consumption. It is inspired by the website of Lionel Dricot (Ploum.net), from whom I borrowed the CSS template for the blog section of this site. It uses the Atkinson Hyperlegible font to improve accessibility.

This website does not collect any data. However, since it is hosted on GitHub, it is possible that Microsoft may collect some data on its side, notably visitors’ IP addresses: (see here).

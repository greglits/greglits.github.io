A propos et mentions légales

Ce site s'inscrit dans une démarche de sobriété numérique. Il est uniquement codé en HTML et CSS et utilise très peu d'images afin de diminuer les ressources utilisées. Il s'inspire du site de Lionel Dricot (Ploum.net) à qui j'ai emprunté le template CSS pour la partie blog de ce site. Il utilise la police Atkinson Hyperlegible afin d'augmenter son accessibilité.

Ce site ne collecte aucune donnée. Il est cependant hébergé sur Github et il est donc possible que Microsoft collecte quelques données de son côté notamment l'adresse IP des visiteurs : (voir ici).

Tous les textes publiés sont mis à disposition sous une licence CC by-nc.
